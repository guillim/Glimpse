Design — Section 3: Character Generation & Grid Layout

How unique creatures are drawn and arranged on screen

Procedural Character Generator

Session ID → Unique Creature
Input: sessionID (UUID string)
Output: CGImage (64–96px, transparent background)

Seed derivation:
  hash(sessionID) → deterministic random number generator
  Same session always produces the same character.

Randomized traits (from seed):
  ┌─────────────┬──────────────────────────────────┐
  │ Body shape  │ Round, Oval, Square-ish, Blob    │
  │ Body color  │ From Pokemon palette:            │
  │             │ Yellow, Red, Blue, Green, Purple, │
  │             │ Orange, Pink, Teal, Brown         │
  │ Eye style   │ Dot, Circle, Anime, Sleepy       │
  │ Ear/horn    │ Pointy, Round, Antenna, None     │
  │ Tail        │ Lightning, Swirl, Flame, None    │
  │ Mouth       │ Smile, Open, Cat-mouth, Line     │
  │ Cheek marks │ Circles, Triangles, None         │
  │ Accent color│ Contrasting belly/ear tips       │
  └─────────────┴──────────────────────────────────┘

Drawing order (Core Graphics):
  1. Body fill (main shape + color)
  2. Accent patches (belly, ear tips)
  3. Eyes (with highlight dot)
  4. Mouth
  5. Ears/horns
  6. Tail
  7. Cheek marks (optional)

Result: ~200 unique combinations
Each session gets a visually distinct creature that persists
for the lifetime of that session.

Adaptive Grid Layout

Grid Scaling Rules

1 session

96px centered

2-4 sessions

80px, 1 row

5-9 sessions

64px, 3-col grid

10+ sessions

48px, 4-col grid

Layout Algorithm
func layoutGrid(count, screenSize):

  // Determine columns based on count
  cols = 1           if count == 1
  cols = count       if count <= 4   (single row)
  cols = 3           if count <= 9
  cols = 4           if count <= 16
  cols = 5           if count > 16

  rows = ceil(count / cols)

  // Character size fills available space with padding
  cellW = screenWidth / cols
  cellH = screenHeight / rows
  charSize = min(cellW, cellH) * 0.55   // 55% of cell for character
  charSize = clamp(charSize, 48, 96)    // min 48px, max 96px
