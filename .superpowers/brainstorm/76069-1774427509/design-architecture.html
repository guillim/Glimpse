Key Components

PokemonScene SKScene subclass. Owns the grid layout, character nodes, empty-state message. Updates from HeadTracker in update(_:) callback.
SessionMonitor Background timer (5s interval). Scans ~/.claude/projects/ for active JSONL sessions. Parses last few lines to determine activity state. Notifies PokemonScene of changes.
CharacterNode SKNode subclass wrapping: body sprite, status icon, label, hello bubble. Handles its own animations (status transitions, hello fade, goodbye fadeout).
CharacterGenerator Pure function: sessionID → CGImage. Uses Core Graphics to draw a unique Pokemon-styled creature from a deterministic seed. Same session always gets same character.
View Swap When switching to/from Pokemon theme, DesktopWindowController swaps its contentView between SCNView and SKView. Other themes unchanged.
