Grid Layout Behavior

How should characters arrange themselves as sessions come and go?

A. Fixed Grid

Characters snap to grid cells. Empty cells stay empty. Predictable, orderly. Characters always in the same slot until their session ends.

B. Organic Scatter

Characters placed at semi-random positions on a "ground plane", like Pokemon in tall grass. More natural and playful. Positions derived from session ID so they're stable.

2 sessions → big & centered

8 sessions → smaller grid

C. Adaptive Grid

Grid that resizes characters based on count. Few sessions = large centered characters. Many sessions = smaller grid. Characters scale to fill available space optimally.
