A

Filler-strip + verb-noun

Strip common filler prefixes ("i want to", "can you", "please", "let's", "could you"), then take the verb + first noun phrase.

dir: background
improve the label
More examples:
"can you fix the bug in SessionMonitor" → fix the bug
"add a countdown timer to the scene" → add a countdown
"please refactor extractTopic" → refactor extractTopic

Pros

  • Simple to implement
  • Predictable output
  • No dependencies

Cons

  • Still truncates — may lose the object
  • Can't handle complex phrasing
B

Keyword extraction (verb + object)

Maintain a list of common action verbs (fix, add, improve, refactor, update, remove, create, implement...). Find the first verb, then grab its object — skip articles and filler.

dir: background
improve label
More examples:
"can you fix the bug in SessionMonitor" → fix bug
"add a countdown timer to the scene" → add countdown timer
"please refactor extractTopic" → refactor extractTopic
"the label is broken when idle" → label broken idle

Pros

  • Concise and meaningful
  • Handles varied phrasing
  • Still pure Swift, no NLP deps

Cons

  • Verb list needs curation
  • Struggles with non-imperative sentences
C

NLTagger (Apple NLP)

Use Apple's built-in NLTagger to part-of-speech tag the message, then extract verbs + nouns automatically. No hardcoded word lists.

dir: background
improve label agent
More examples:
"can you fix the bug in SessionMonitor" → fix bug SessionMonitor
"add a countdown timer to the scene" → add countdown timer scene
"please refactor extractTopic" → refactor extractTopic
"the label is broken when idle" → label broken idle

Pros

  • Handles any phrasing — no word lists
  • Built into macOS, no external deps
  • Extracts real meaning, not just position

Cons

  • NLTagger can be inconsistent with code terms
  • Slightly more code complexity
